# Electrodomésticos :

Electrodomésticos es una pequeña aplicación que permite ingresar distintos
datos correspondientes a electrodomésticos, entre estos esta lavadoras y 
televisor. Una vez ejecutada la clase principal se desplegara un listado 
de los Electrodomésticos previamente registrados y se desplegara los precios 
específicos para cada producto según su categoría.
